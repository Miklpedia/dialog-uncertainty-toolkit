# Audio Data

This directory should contain short English audio samples
(10–30 seconds each) used by the notebooks. **No audio is committed to
this repository.**

## Recommended public sources

- **LibriSpeech mini** — https://www.openslr.org/12  
  Clean read English, public domain.
- **Mozilla Common Voice** — https://commonvoice.mozilla.org/  
  Crowdsourced English (and other languages), CC0.
- **DailyTalk (HuggingFace)** — short conversational utterances.
- **Synthetic TTS** — generated locally with `edge-tts` or `gTTS` for
  controlled experiments.
